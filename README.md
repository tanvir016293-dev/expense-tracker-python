# 💸 Expense Tracker (Python)

A simple console-based Expense Tracker application built using Python.

## 🚀 Features

- Add new expenses
- View all expenses
- Calculate total spending
- Store expenses using Python dictionaries and lists
- Easy-to-use menu-driven interface

## 🛠 Technologies Used

- Python 3

## ▶️ How to Run

1. Clone this repository

```bash
git clone https://github.com/tanvir016293-dev/expense-tracker-python.git
```

2. Run the project

```bash
python expense_tracker.py
```
